THE WEB

The web is the communication between servers, the internet and your computer.
The computer sends a request to the internet and from the internet to the servers in the form of HTML.
Then information is transfered back to the internet and subsequently to your computer.

HTML

HTML - Hypertext Markup Language.
HTML is the protocal to talk to the servers.
HTML documents form the majority of the content on the web.

    HTML is made up of:
  • Text content
  • Markups - code that alter the arrangement and visuals
  • References to other documents, videos, pictures etc.

TAGS

Tags- part of the code in a markup. It is included in what is known as the element.
Element and Tag are slightly interchangeable.
The element is the complete code for the markup.
Tags are written with a beginning tag and an ending tag both surrounded by " <>".
Some tags are void tags which means they don't need an ending.
Last but not least there are too many tags to memorize so prgrammer look stuff up!

    Some examples of tags are:
  • inserting an image
  • or a link "hidden in text".

Inline vs Block Elements

HTML elements are either inline or block.
Block elements form an "invisible box" around the content inside of them whereas inline does not.
